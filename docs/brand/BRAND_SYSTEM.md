# Brand System

## Source of truth

The human-facing source of truth is the CSO Group Master Brand System in SharePoint:

`CSO Family / S Drive / CSO Group / Brand and Marketing / 00 Master Brand System`

Use that library for approved logos, favicons, colours, fonts, PDF examples, social examples, and any signature asset.

## Rules for product work

- Read this file before visual changes.
- Preserve the distinction between CSO, Little Miracles, and Camica. They are related brands, not interchangeable skins.
- Use only assets, colours, and fonts that are explicitly approved in the shared library or already established by the relevant product.
- Do not infer or invent brand values from screenshots, old files, or another brand's CSS.
- If an approved choice is missing, preserve the existing product styling and flag the decision needed rather than redesigning.
- Keep visual changes limited to the requested screen or component.
- When a brand decision is approved, update the SharePoint master first, then update this file if the coding guidance changes.
